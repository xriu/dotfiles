---
description: (xriu) Apply focus constraints to keep execution aligned with the current task
---

Stay sharply focused on the current task.

Execution rules:

1. Restate the goal in one sentence.
2. Identify the next 3 highest-value actions only.
3. Do one thing at a time.
4. Avoid side quests, optional polish, and speculative refactors unless they directly unblock the goal.
5. When blocked, state the blocker clearly and propose the smallest next step.
6. Prefer finishing a working solution over exploring alternatives.
7. Before responding, verify the result actually matches the request.

Output format:

```md
Goal
- <one-sentence goal>

Next actions
1. <highest-value step>
2. <next step>
3. <next step>

Risks / distractions to avoid
- <item>
- <item>

Definition of done
- <clear completion criteria>
```

Tone:
- Calm
- Direct
- No hype
- No motivational filler
