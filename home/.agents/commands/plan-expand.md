---
description: (xriu) Expand plans into detailed, delegation-ready execution steps
---

Take the current plan, roadmap, or todo list and expand it into detailed, actionable steps that are easy to execute or hand off to other agents.

## Goals

- increase clarity
- reduce ambiguity
- make progress trackable across long tasks or compactions
- produce steps that can be delegated with minimal extra explanation

## Process

1. Identify the major phases or workstreams.
2. Break each phase into concrete tasks and sub-tasks.
3. Add dependencies, prerequisites, and checkpoints where useful.
4. Keep the list detailed enough to act on, but avoid filler.
5. Prefer high-value sequencing over sheer number of items.

## Output format

```md
Summary
- <what the plan covers>
- <main objective>

Expanded plan
1. Phase: <name>
   - Goal: <what this phase achieves>
   - Steps:
     1. <specific task>
     2. <specific task>
     3. <specific task>
   - Deliverables:
     - <artifact / result>
   - Dependencies / risks:
     - <if any>

2. Phase: <name>
   ...

Handoff notes
- <what another agent or teammate would need to know>
```

Guidance:
- Aim for at least 12-20 meaningful tasks when the scope is substantial.
- Add more detail inside each phase instead of padding the list with low-value items.
- If the plan is already detailed enough, tighten and organize it instead of expanding blindly.
